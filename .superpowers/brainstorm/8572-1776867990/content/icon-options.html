ทำไมถึงเป็น Abstract Layers?

  • Depth: การใช้ความโปร่งแสงซ้อนกันสร้างมิติที่ทำให้ Icon ดูไม่แบน
  • Focus: ตัวนาฬิกาทรายถูกดัดแปลงให้เป็นรูปทรงเลขาคณิตที่ดูเรียบง่ายแต่จำง่าย
  • Modern: เข้ากับ UI ของทั้ง macOS (Sonoma/Sequoia) และ Windows 11
